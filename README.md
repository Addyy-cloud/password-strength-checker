A Python-based tool that evaluates password strength based on
security best practices such as length, character diversity, and symbols.

## Features
- Checks minimum password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Provides real-time feedback and suggestions
- Classifies passwords as Weak, Medium, or Strong

## Tech Stack
- Python
- Regular Expressions (re module)

## How It Works
The program analyzes user input using pattern matching and
returns both the strength level and improvement suggestions.
